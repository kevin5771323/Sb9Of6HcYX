# 图书管理系统 java1052

## 项目概述

图书管理系统是一款旨在帮助图书馆管理员高效管理图书资源的应用。系统采用前后端分离的技术架构，前端使用 Vue.js，后端采用 Java Spring Boot 和 Python Flask 框架，实现实时同步京东新书数据，方便图书馆快速上架最新图书。

## 技术栈

- 前端：Vue.js
- 后端：Java Spring Boot、Python Flask

## 功能模块

- 实时同步京东新书数据
- 图书信息管理
- 图书借阅管理
- 用户管理
- 系统管理

## 系统角色

- 系统管理员
- 图书管理员
- 读者

## 运行环境

- 前端：现代浏览器
- 后端：Java 8+、Python 3.6+

## 部署步骤

1. 部署后端服务（Java Spring Boot、Python Flask）
2. 部署前端服务（Vue.js）
3. 配置数据库
4. 启动系统

## 目录结构

```
项目根目录/
├── backend     # 后端代码
│   ├── java    # Java Spring Boot 代码
│   └── python  # Python Flask 代码
└── frontend    # 前端代码（Vue.js）
```

## 核心亮点

- 实时同步京东新书数据，快速上架最新图书
- 基于前后端分离架构，易于维护和扩展
- 丰富的功能模块，满足图书馆日常管理需求
- 支持多种系统角色，实现不同权限管理
- 灵活的部署方式，适应不同运行环境
